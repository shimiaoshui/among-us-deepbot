Among Us DeepBot 0.9.11 + The Other Roles v4.6.0 汉化大厅身份与会议隔离包

适用：同一房间的所有玩家安装完全相同的本包。
本包包含明确标记为修改版、以 GPLv3 发布的 TheOtherRoles.dll；完整对应源码和许可证见同一 GitHub Release。运行 Install-DeepBot.cmd 安装。
房主运行 Configure-DeepBot-Key.cmd，并在 TOR 等待房间设置 AI Bot 数量（1-8），还能分别设置名字、颜色、服装和名牌；客端无需密钥且不会自行生成 Bot。
重名 AI 会自动加编号以避免身份文字重叠造成误解。每个玩家最多拥有一个主体职业，只有附加职业可以叠加。
所有职业数量、冷却、任务和最终胜负以房间/TOR 设置为准。
本包不包含游戏、BepInEx 或任何 API 密钥。
